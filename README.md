# Cnblogs-Theme-Xhanglog
博客园皮肤，在作者BNDong基础上修改，原地址https://github.com/BNDong/Cnblogs-Theme-SimpleMemory

博客设置
1. 设置博客皮肤
博客皮肤：SimpleMemory

2. 设置页面定制CSS代码
CSS代码: base.min.css 拷贝此文件代码至页面定制CSS代码文本框处。
   base.min.css 的修改参考 /src/style/base.css。博客设置请使用压缩版本，直接使用 base.css 会字符超限！

3. 禁用模板默认CSS
选中页面定制CSS代码文本框下面的禁用模板默认CSS。

配置完成保存即可成功应用博皮！
